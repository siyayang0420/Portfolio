CourtOn is a mobile application created by students at the British Columbia Institute of Technology. The objective of the app is to create a platform for badminton players to host a game with a self-created group or to join other organizer’s groups at any of the badminton centres in Vancouver’s Lower-mainland.

UI/UX Solution

The goal for this application was to enhance the competitive and leisure experience this app brought to the users while making it easy to navigate. We chose a vivid and energetic colour scheme for the app, along with the original hand-drawn illustration to make the user interface more attractive and interesting. Users could easily navigate between joined and created groups by the information hierarchy we designed for users. The skill-level was helpful in a way that if the user was looking for a suitable group to play, and the skill-level indicators would be display altogether with other users so that users would able to know whenever joined a group.
